Unity Asset Transform

Pipeline automation, industrial licensing, and CAD ingestion at scale

What Is Unity Asset Transform?

Unity Asset Transform (UAT) is Unity's industrial-grade solution for converting CAD and other complex 3D data into Unity-optimized assets. It's part of Unity's industrial/enterprise offerings.

🧠 The Core Value

UAT solves the CAD-to-game problem at scale. Instead of manually exporting, fixing, and optimizing each file, you define a pipeline that processes data automatically. Essential when you have hundreds of clients submitting data.

Core Features

  • Tessellation Control: Convert NURBS with configurable quality (chord deviation, angle tolerance)
  • Mesh Decimation: Reduce polygon count with quality thresholds
  • LOD Generation: Automatically create multiple detail levels
  • Hierarchy Preservation: Keep assembly structure or flatten
  • Material Mapping: Extract or remap CAD materials to Unity materials
  • Metadata Extraction: Preserve CAD properties as Unity metadata

UAT API Usage (Conceptual)

using Unity.AssetTransform;

public class CADProcessor : MonoBehaviour
{
    public void ProcessCADFile(string inputPath)
    {
        // Create processor with settings
        var processor = new CADProcessor();
        
        processor.TessellationSettings.ChordDeviation = 0.1f;
        processor.OptimizationSettings.TargetTriangles = 50000;
        processor.LODSettings.Levels = 3;
        
        // Process and get result
        var result = processor.Import(inputPath);
        
        // result contains meshes, hierarchy, materials, metadata
        InstantiatePrefab(result.RootObject);
    }
}

Note: This is conceptual — actual UAT API may differ. Consult documentation.
