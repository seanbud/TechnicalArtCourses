System Architecture Map

The "Big Picture" view of the entire pipeline. Visualize this during your interview.

1. ACQUISITION (The Source) Vicon / Shogun
Math: SVD (Singular Value Decomposition)
Algorithm: Rigid Body Solving
Tech: Vector3 Class
Problem: Marker Swaps
"Garbage In, Garbage Out. We use SVD to mathematically minimize the error between the cloud of points and the rigid bone."
2. INGESTION (Perforce) Helix Core / P4Python
Admin: p4 typemap (Exclusive Locking)
Security: p4 protect (IP Isolation)
Automation: Server-Side Triggers
Code: P4.connect()
"We use Perforce because binaries cannot be merged. Locking is our safety net against lost work."
3. TOOLING (PySide) Python / Qt / Maya
Constraint: The GIL (Global Interpret Lock)
Pattern: Model-View-Controller (MVC)
Solution: QThread & Signals
UX: "The Cockpit Philosophy"
"I separate Logic from UI so the tool can run Headless on a server later. Never touch Maya from a thread."
4. AUTOMATION (CI/CD) GitLab CI / Docker
Config: .gitlab-ci.yml
Env: Docker Container (Clean Room)
Mode: Headless (mayapy -batch)
infra: GPU Runners
"It works on my machine is a lie. Docker ensures every build runs in an identical environment."
5. DISTRIBUTION (The Farm) Deadline / AWS
Scale: Horizontal (100+ Nodes)
Concept: Idempotency (Safe Retries)
Code: JobInfo / PluginInfo files
Structure: Directed Acyclic Graph (DAG)
"We use Chunking to amortize the startup tax (opening Maya) across multiple frames."
6. DATA STRATEGY (Lifecycle) SQL / Isilon / S3
Storage: Tiered (Hot -> Warm -> Cold)
Validation: MD5 Hashing
Tracking: SQLite / Asset Database
Process: The "Sweeper" Bot
"Moving data without hashing is gambling. A single bit flip can corrupt a 50GB file."
7. SYSTEM DESIGN (Leadership) Architecture / Culture
Pattern: Decoupling (Queues)
Design: "The Proxy Pattern" (Multi-site)
Behavior: Blameless Post-Mortems
Method: The STAR Method
"A Senior TA designs for failure. A Junior TA designs for the happy path."
