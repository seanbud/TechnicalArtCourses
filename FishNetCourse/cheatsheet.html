3. CSP Boilerplate: Movement Networking

Copy-paste this structure for any predicted movement script (Player, Boat, Plane). Ensure the simulation logic is separated into a "Pure" class.

C#: Production-Ready CSP Template

using FishNet.Object;
using FishNet.Object.Prediction;
using FishNet.Transporting;
using UnityEngine;

public class NetworkedActor : NetworkBehaviour 
{
    [SerializeField] private ActorSimulation _sim; // Pure simulation logic

    public override void OnStartNetwork() {
        base.OnStartNetwork();
        TimeManager.OnTick += OnTick;
    }

    public override void OnStopNetwork() {
        base.OnStopNetwork();
        if (TimeManager != null) TimeManager.OnTick -= OnTick;
    }

    private void OnTick() {
        if (IsOwner) {
            MoveData data = BuildInput();
            Replicate(data);
        }
    }

    [Replicate]
    private void Replicate(MoveData data, ReplicateState state = ReplicateState.Invalid, Channel channel = Channel.Unreliable) {
        // MUST use TickDelta, not deltaTime
        _sim.Simulate(data, (float)TimeManager.TickDelta);
    }

    public override void CreateReconcile() {
        MoveState state = _sim.GetState(); // Capture everything that affects physics
        Reconcile(state);
    }

    [Reconcile]
    private void Reconcile(MoveState state, Channel channel = Channel.Unreliable) {
        _sim.SetState(state); // Snap and prepare for replay
    }
}

6. Common Gotchas & Fixes

  • Character Vibrating? You used Time.deltaTime instead of TimeManager.TickDelta in [Replicate].
  • Jump Height Inconsistent? You aren't reconciling your jumpTimer or verticalVelocity.
  • RPC Not Firing? Ensure you have a NetworkObject on the GameObject. If it's a ServerRpc, ensure the caller is the Owner.
  • SyncVar Hook Not Firing? Hooks only fire when the value changes. If you set Hp = 10; and it was already 10, nothing happens.
  • Physics Desync? Set Physics.simulationMode = SimulationMode.Script in your bootstrap.

7. Bandwidth Optimization Tips

  • Use Channel.Unreliable for movement data. If a packet is lost, the next one arrives 16ms later anyway.
  • Compress your structs. Use short or half for rotations if you don't need 32-bit float precision.
  • Cull Observers. Use SceneCondition or DistanceCondition so players in a 2D co-op setting don't receive updates for objects 3 scenes away.
